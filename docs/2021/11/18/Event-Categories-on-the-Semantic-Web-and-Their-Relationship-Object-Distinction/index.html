Abstractの訳

イベントはオントロジカルにモデル化することの繰り返されるトピックである．そして，セマンティックウェブのオントロジー言語であるOWLでのコード化の多様さはとても大きい．私たちは，基盤的なものとセマンティックウェブのものの両方の観点からイベントモデリングのアプローチについて軽い比較調査を行う．そして，「イベント」と共通して呼ばれるものの4つのカテゴリを一時的に作りなおす．軽量のオントロジカルな背景をモデリングする言語であるPUROによって表されるように，カテゴリを作ることの実体の部分はオブジェクトと関係の区別が明確であるべきである．
